# Strategic Analysis Lab

AI-powered strategic analysis tool for autonomy and robotaxi scale questions.

## Features

- Structured 9-section analysis format
- Real-time public data integration
- Clickable example questions
- Waymo-specific analysis (updated from Company B)

## Quick Start

1. Enter your strategic question
2. Click "Analyze" for detailed analysis
3. Explore examples for inspiration

## Examples

- "Can Waymo become a $1T company by 2035?"
- "City fleet capacity: SF 15K robotaxis"
- "Market scale: 500M trips/week by 2040"

Built with Next.js, OpenAI, and Tavily API.
